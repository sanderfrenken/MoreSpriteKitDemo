# MoreSpriteKitDemo

This repository showcases the various functionalities that can be used from [MoreSpriteKit](https://github.com/sanderfrenken/MoreSpriteKit).

All sources are written in Swift.

### Requirements

- iOS 10.3+
- Xcode 10.1+
- Swift 5+
